# CodeAlpha_Social-media-platform

A simple social media web application built with **Django** where users can register, log in, create posts, comment on posts, and follow other users.

## 📌 Features

- User Registration
- User Login & Logout
- Create Posts
- Upload Images with Posts
- Comment on Posts
- Follow / Unfollow Users
- Responsive Bootstrap UI
- Django Admin Panel

---

## 🛠 Tech Stack

- **Backend:** Django
- **Frontend:** HTML, CSS
- **Database:** SQLite3
- **Language:** Python 3

---


## Database Tables

- User (Django Authentication)
- Profile
- Post
- Comment
- Follower
